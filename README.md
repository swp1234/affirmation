# 일일 긍정 확언 카드 ✨

매일 새로운 긍정 확언으로 하루를 시작하세요.

## 기능

- 📖 **100개의 긍정 메시지** - 5개 카테고리 (자존감, 동기부여, 감사, 관계, 성공)
- 🎲 **랜덤 카드 뽑기** - 매일 새로운 메시지
- ❤️ **즐겨찾기** - 마음에 드는 확언 저장
- 📊 **통계 추적** - 읽은 카드 수, 연속 일수
- 🌓 **다크/라이트 모드** - 눈에 편안한 테마
- 📱 **PWA 지원** - 모바일 홈 화면에 설치 가능
- 🔗 **공유 기능** - 좋은 메시지 공유하기

## 기술 스택

- HTML5, CSS3, Vanilla JavaScript
- Progressive Web App (PWA)
- LocalStorage for data persistence
- Responsive Design

## 로컬 실행

```bash
# 간단한 HTTP 서버 실행
python -m http.server 8000

# 브라우저에서 접속
http://localhost:8000
```
